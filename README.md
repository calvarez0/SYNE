# SYNE

neuroevolution through genome fusion instead of mutation. networks grow by merging with each other, not by random changes.

## folder structure

- **syne/** - original implementation
- **syne_v2/** - second iteration with refinements
- **syne_v5/** - latest version (most up-to-date)
- **research/** - benchmarking scripts and experimental results. includes comparisons with NEAT, performance analysis, and data/figures
- **examples/** - simple demo scripts showing how to use SYNE (xor problem, pole balancing, mutation comparisons)

## version differences

each version folder (syne, syne_v2, syne_v5) contains a complete implementation with slight variations in the algorithm. v5 is the current working version with the most recent improvements.
